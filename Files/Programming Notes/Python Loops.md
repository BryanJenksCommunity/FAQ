# Python Loops

## While Loop

```python
i = 1
while i <= 10:
    print(i)
    i+=1
```

## For Loop

```python
for letter in "giraffe academy":
    print(letter)

friends = ["Jim", "Suzy", "Kevin"]
for name in friends:
    print(name)

for index in range(10):
    print(index) # prints 1-9 not including 10 so always increment upwards by 1

```
